getLineName¶

Επιστρέφει το όνομα της γραμμής. Για να χρησιμοποιηθεί χρειαζόμαστε και την παράμετρο linecode που μπορούμε να την βρούμε αν φιλτράρουμε τον αριθμό του λεωφορείου που χρειαζόμαστε από τις δράσεις webGetLines και webGetLinesWithMLInfo

Για παράδειγμα ( linecode = 1220 ) που είναι η γραμμή ΚΑΤΕΧΑΚΗ-ΚΥΨΕΛΗ , έχει διαδρομές ΚΑΤΕΧΑΚΗ-ΚΥΨΕΛΗ και ΚΥΨΕΛΗ-ΚΑΤΕΧΑΚΗ.

Api Endpoint¶

http://telematics.oasa.gr/api/?act=getLineName&p1=linecode
