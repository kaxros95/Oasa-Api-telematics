webRoutesForStop¶

Επιστρέφει πληροφορίες σχετικά με μια στάση. Χρειαζόμαστε και μια παράμετρο που είναι το stopcode δηλαδή ο κώδικας μιας στάσης. Για να τον βρούμε μπορούμε να χρησιμοποιήσουμε την webGetStops. Στο παράδειγμα εμφανίζονται τα απότελέσματα του webRoutesForStop της στάσης ΛΑΜΠΡΙΝΗ ( stopcode = 10334 )

Api Endpoint¶

http://telematics.oasa.gr/api/?act=webRoutesForStop&p1=stopcode
