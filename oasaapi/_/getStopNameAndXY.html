getStopNameAndXY¶

Επιστρέφει το όνομα της στάσης και τις συντεταγμένες της. Χρειαζόμαστε και μια παράμετρο που είναι το stopcode δηλαδή ο κώδικας μιας στάσης. Για να τον βρούμε μπορούμε να χρησιμοποιήσουμε την webGetStops. Στο παράδειγμα εμφανίζονται τα απότελέσματα του getStopArrivals της στάσης ΤΕΡΨΙΘΕΑ ( stopcode = 400053 )

Api Endpoint¶

http://telematics.oasa.gr/api/?act=getStopNameAndXY&p1=stopcode

Response¶

[
{
"stop_descr":"\u03a4\u0395\u03a1\u03a8\u0399\u0398\u0395\u0391",
"stop_descr_matrix_eng":"TERPSITHEA",
"stop_lat":"37.9405163",
"stop_lng":"23.645139"
"stop_heading":"39",
"stop_id":"400053"
}
]
